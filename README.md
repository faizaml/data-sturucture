# 🖥️ Data Structures & Algorithms

This repository is dedicated to documenting university assignments, coding practices, and implementations of various data structures using the **C** programming language.

## 📂 Project Structure
* `Trees-DSA/` - Binary Tree implementations and search tree algorithms.
* `FinalProject-DSA/` - Main final project for the Data Structures course.
* `Heap-DSA/` - Priority queue implementations using Heap.
* `RBToperation-DSA/` - Exploration of operations on Red-Black Trees.

## 🛠️ Development Environment
* **Language:** C
* **Editor:** VS Code (macOS)
* **Compiler:** GCC / Clang
